## Overview

**Indo Investor Infra World** is a high-performance real estate investment platform showcasing RERA-verified residential plots, premium commercial offices, and co-working pod investments across India's highest-growth corridors.

The platform is engineered as a **pure static SPA** вҖ” all data lives in the source code, routing is handled client-side via `HashRouter`, and the entire site is built and deployed automatically to **GitHub Pages** via a single GitHub Actions workflow. No paid hosting is ever required.

## Tech Stack

| Layer | Technology | Role |
|:------|:-----------|:-----|
| **UI Framework** | React 18 + TypeScript 5 | Component architecture & full type safety |
| **Build Tool** | Vite 5 | Lightning-fast dev server, optimised production bundles |
| **Styling** | Tailwind CSS 3 + custom tokens | Utility-first responsive design with design-system tokens |
| **Components** | shadcn/ui + Radix UI | Accessible, headless UI primitives |
| **Animation** | Framer Motion 11 | Page transitions, scroll reveals, micro-interactions |
| **Routing** | React Router 6 (HashRouter) | Client-side routing вҖ” no server config needed |
| **State** | TanStack Query 5 | Async state & data fetching infrastructure |
| **Icons** | Lucide React + React Icons | Consistent icon library |
| **Hosting** | GitHub Pages + GitHub Actions | Free, automated, zero-config static deployment |
